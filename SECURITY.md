# Security Policy

| Versión | Soporte |
|---|---|

## Reportar una vulnerabilidad

Si encuentras una vulnerabilidad en este proyecto, abre un **Issue privado** o contáctame directamente en GitHub: [@robinsons1](https://github.com/robinsons1)

Este es un proyecto personal/educativo. Se intentará responder en un plazo de 7 días.
